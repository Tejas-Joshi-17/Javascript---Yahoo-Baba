Assignment Operators :-


    +------------------+----------------+-------------------+
    |      Operator    |    Example     |     Same As       |
    |------------------+----------------+-------------------+
    |         =        |    x  = y      |     x = y         |
    |        +=        |     x += y     |    x = x + y      |
    |        -=        |     x -= y     |    x = x - y      |
    |        *=        |     x *= y     |    x = x * y      |
    |        /=        |     x /= y     |    x = x / y      |
    |        %=        |     x %= y     |    x = x % y      |
    |       **=        |     x **= y    |    x = x ** y     |
    +------------------+----------------+-------------------+ 

Note :-  Assignment Operators are only applicable to 'number' datatype.
