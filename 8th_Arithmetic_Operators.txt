Arithmetic Operators :-

    +----------------------+-----------------------------------+
    |      Operator        |         Description               |
    |----------------------+-----------------------------------+
    |        +             |          Addition                 |
    |        -             |        Substraction               |
    |        *             |       Multiplication              |
    |        **            |       Exponentiation              |
    |        /             |           Division                |
    |        %             |      Modulus (Remainder)          |
    |        ++            |          Increment                |
    |        --            |          Decrement                |
    +----------------------+-----------------------------------+ 

Note :-  Arithmetic Operators are only applicable to 'number' datatype.
