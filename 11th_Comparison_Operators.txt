Comparison Operators :-

    +----------------------+-------------------------------------------------------+
    |      Operator        |         Description                                   |
    |----------------------+-------------------------------------------------------+
    |       ==             |  Equal to (not check data type only value check)      |
    |       ===            |  Equal value and equal data type                      |
    |       !=             |  Not equal (not check data type only value check)     |
    |       !==            |  Not equal value & not equal data type                |
    |        >             |  Greater then                                         |
    |        <             |  Less than                                            |
    |        >=            |  Greater than or equal to                             |
    |        <=            |  Less than or equal to                                |
    +----------------------+-------------------------------------------------------+
